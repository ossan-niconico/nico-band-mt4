# nico-band-7TF（MT4版）

MT4用インジケーター。1分足〜日足の7タイムフレームの
ATRトレンドバンドを1枚のチャートに同時表示します。

## ファイル

- `nico-band-7TF.mq4` — インジケーター本体

## インストール方法

1. `nico-band-7TF.mq4` をダウンロード
2. MT4メニュー →「ファイル」→「データフォルダを開く」
3. MQL4 → Indicators フォルダに貼り付け
4. MT4を再起動（またはF7でコンパイル）
5. ナビゲーター →「nico-band-7TF」をチャートにドラッグ
